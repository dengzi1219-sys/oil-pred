🛰️ Oil-Pred v3.2 | 深度量化决策指挥终端
Oil-Pred 是一款基于 Python 开发的高级金融量化推演工具。不同于传统的静态技术分析，本系统采用了“人机耦合”的逻辑，允许决策者在数学回归的基础上，注入地缘政治风险溢价和宏观战术修正，实现对国际黄金（COMEX）与原油（NYMEX）市场的极端行情模拟。

🌟 核心亮点
人机耦合决策： 引入“黑天鹅恐慌溢价”滑块，模拟战争、通牒等极端情绪对价格的非线性冲击。

多因子稳态引擎： 底层集成均值回归（Mean Reversion）、RSI 动能耗竭、季节性周期以及美元指数（DXY）天敌打压逻辑。

动态概率云图： 基于历史与推演数据的双重布林带（Bollinger Bands）置信区间，直观展示预测结果的波动风险。

时光机回测系统： 支持“样本外”盲测，可自由穿越至历史任意时点，验证战术参数的拟合效能。

自引导运行： 深度整合 Streamlit 引导模块，支持 VS Code 环境下一键启动 Web UI。

🧠 量化逻辑架构
系统并非简单的趋势线性外推，而是构建了一套“物理级”修正体系：

多项式趋势拟合： 采用二阶多项式回归（Polynomial Regression）捕捉市场惯性。

成交量加权： 引入 Volume Weighting 逻辑，放量行情在模型中具有更高的权重。

对数情绪衰减： 模拟恐慌情绪从“爆发”到“审美疲劳”的自然衰减过程。

价值锚点牵引： 自动锁定 150 日价格中位数，当价格极端偏离时产生强大的均值回归拉力。

物理底线保护： 动态计算历史支撑位，防止预测曲线在极端参数下出现非理性塌缩。

🛠️ 技术栈
数据源： AkShare (实时对接国内/国际财经接口)

核心算法： Scikit-learn (Linear Regression)

数据处理： Pandas / Numpy

可视化： Plotly (交互式动态图表)

交互框架： Streamlit
